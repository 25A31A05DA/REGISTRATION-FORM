# REGISTRATION-FORM

___

A simple Registration Form built as part of my freeCodeCamp journey.  
The goal of this project was to practice Semantic HTML (fieldset, labels) and improve visual hierarchy using CSS.  
I focused on building a clean layout, structured grouping, and a modern light-themed UI.

___

## What this project includes

- Clean Semantic HTML  
- Structured form layout using fieldset  
- Responsive width control  
- Modern light color theme  
- Styled submit button  
- Good use of CSS

___

## What I Learned

### Semantic HTML  
Used `<fieldset>` and `<legend>` to logically group related inputs.  
Applied proper `<label>` associations for accessibility and better structure.  
This improves readability and makes the form more accessible for assistive technologies.

### Accessibility & Usability  
Labels increase clickable area for inputs like radio buttons and checkboxes.  
Required fields help guide user interaction clearly.  
Focus states improve keyboard navigation.

### UI & Visual Hierarchy  
Learned how color contrast creates separation between sections.  
Used subtle borders and accent colors to guide attention.  
Designed a clear call-to-action button that stands out.
